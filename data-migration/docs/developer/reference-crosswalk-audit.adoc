= Reference: crosswalk audit
:toc: left

Do the committed crosswalks still agree with the *current* PROD source MySQL?
This page is the live inventory of unmapped legacy codes, orphaned mappings,
label and snapshot drift, and missing source tables/columns -- the drift the
`sql/04_crosswalks/*_check.sql` guards cannot see because they run only against
`mysql_raw`. The body is generated by `scripts/crosswalk_audit.py` against the
live source; run `make crosswalk_audit` to refresh it. For the static
target-column view of crosswalk coverage, see
xref:reference-derivability-audit.adoc[the derivability audit].
