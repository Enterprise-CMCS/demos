= Migration plan: branch decision tree and status
:toc: left

This page is the canonical tracker for the cross-repo migration work that
mines the sister DEMOS application repo (`Enterprise-CMCS/demos`) to advance
the PMDA MySQL -> DEMOS PostgreSQL (`demos_app`) migration. Each "branch" is a
resolved decision plus its follow-up work. The xref:canonical-spec.adoc[canonical
specification] carries the full narrative; this page is the index, the status
board, and the dependency map.

== How to read this page

* The <<decision-tree,decision tree>> lists every branch, its resolution, the
  DEMOS asset that resolved it, and its current status.
* The <<critical-path,critical path>> shows which branches gate which.
* The <<provenance,decision provenance>> table says where on disk each
  decision is recorded, so this page never becomes the only source of truth.

Branch numbering is stable: cite "Branch 8" in commits and reviews.

[[critical-path]]
== Critical path

Branch 1 is the root dependency: the refreshed Prisma contract gates the BN
fill domain (Branch 6), the whitespace-CHECK coupling (Branch 13), and the
`30_constraints` / `99_parity` deltas. Branch 3 provides the parity table that
Branch 8 fills; Branch 8 feeds the Branch 5 loader, which feeds Branch 7.
Branch 4 runs in parallel with Branch 1. Branch 2's transforms are authored
against the source-of-truth trigger contract but snapshot the *live* tables, so
they only produce rows once Branch 5 loads `demos_app.*` (B5 -> B2 below).

[mermaid]
....
flowchart TD
  B1[B1 Contract refresh] --> B6[B6 BN fill domain]
  B1 --> B13[B13 functions.sql trim + invariants]
  B1 --> CP[30_constraints + 99_parity deltas]
  B1 --> B5[B5 BN target load - 20_app]
  B4[B4 Crosswalk confidence] -.parallel.-> CP
  B3[B3 JSONB disposition] --> B8[B8 BN source + staging]
  B8 --> B5
  B6 --> B5
  B5 --> B7[B7 BN revalidation at flip]
  B5 --> B2[B2 History - DEMOS-owned, not migrated]
  B13 --> CP
....

[NOTE]
====
Solid arrows are hard dependencies; the dashed edge marks Branch 4 as
parallelizable with the Branch 1 critical path.
====
