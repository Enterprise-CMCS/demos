= Reference: drop list

Tables we explicitly do *not* migrate from MySQL. The SME review
ensures we are not silently losing live data.

== Authoritative list

`pgloader/drop_list.txt` -- one table name per line, `#` comments
allowed. Rendered into pgloader's `EXCLUDING TABLE NAMES MATCHING`
clause by `migration/phases/load_full.py`.

== Rationale

Live contents of `reports/narrative/drop_list.md`. SME review confirms no table
on the drop list holds live data the new app needs; complete the
sign-off block in the file when satisfied.
